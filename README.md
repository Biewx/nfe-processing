# NF-e Intelligence

Plataforma para análise de compras baseada em XML de NF-e (modelo 55).

O objetivo é transformar notas fiscais eletrônicas em informações estratégicas, como:

- histórico de preços;
- evolução de custos;
- comparação entre fornecedores;
- análise de produtos;
- dashboards de compras.

Tecnologias:
- NestJS
- Prisma
- PostgreSQL
- TypeScript
